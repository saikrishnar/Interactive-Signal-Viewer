Interactive Signal Viewer


Task :

Build a web page that has a login screen. The login screen must have username and password fields. On entering a correct username/password combination, the user is taken to a page where is displayed an interactive signal viewer. The signal viewer must show the data which is attached below. The graph must be scrollable and zoomable. Each row of the data consists of a single time step. The value of each row consists of 2 floating point numbers each representing the amplitude of one channel of the signal at that time. Both channels must be plotted and displayed.
